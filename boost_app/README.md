

## 一个完整的字符串操作库应该具备以下功能：
- 字符串长度：获取字符串的长度，即字符的数量。
- 字符串拷贝：将一个字符串复制到另一个字符串中。
- 字符串连接：将两个字符串连接成一个新的字符串。
- 字符串比较：比较两个字符串的大小关系，判断是否相等、大于或小于。
- 字符串搜索和替换：在字符串中搜索指定的子串，并进行替换。
- 字符串分割：将一个字符串按照指定的分隔符分割成多个子串。
- 字符串查找：在字符串中查找指定的子串，并返回其位置。
- 字符串截取：截取字符串的一部分作为一个新的子串。
- 字符串转换：将字符串转换为其他类型的数据，如整数、浮点数等。
- 字符串大小写转换：将字符串中的字符转换为大写或小写形式。
- 字符串删除、插入和替换：删除、插入或替换字符串中的指定字符、子串或片段。
- 字符串格式化：根据指定的格式将数据插入到字符串中。
- 字符串解析和转义：解析字符串中的特定格式，并进行转义处理。
- 字符串处理性能优化：针对字符串处理的性能进行优化，如避免频繁申请内存等。
- 字符串编码和解码：进行字符串的编码和解码操作，如 base64 编码等。
